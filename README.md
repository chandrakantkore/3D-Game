# 🎮 3D Game

![Unity](https://img.shields.io/badge/Unity-3D-black?logo=unity)

A modern **3D game** developed using **Unity 3D** and **C#**, featuring immersive gameplay, interactive environments, smooth player controls, and realistic graphics. This project demonstrates core game development concepts such as physics, animations, collisions, UI, and scene management. 0

# 🎯 Features

- 🎮 Smooth Player Controls
- 🌍 Interactive 3D Environment
- 🎥 Third-Person/First-Person Camera
- 💥 Collision Detection
- 🎵 Background Music & Sound Effects
- ⚡ Optimized Performance
- 🏆 Score & Game Over System
- 📱 Responsive UI

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/chandrakantkore/3D-Game.git
```

### Open Project

1. Open **Unity Hub**
2. Click **Open Project**
3. Select the cloned folder
4. Press **Play** inside Unity
